# ASP.NET Razor Calculator
A simple web-based calculator built using C# and ASP.NET Razor Pages in Visual Studio.

## Features
* Basic Arithmetic (Addition, Subtraction, Multiplication, Division)
* Clean UI using Bootstrap
* Server-side logic using C#

## How to Run
1. Clone this repository.
2. Open the `.sln` file in Visual Studio.
3. Press `F5` to run the project in your browser.
